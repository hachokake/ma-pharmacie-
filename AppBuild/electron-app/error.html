Impossible de se connecter

L'application ne peut pas se connecter au serveur Django.

Vérifiez que :

  • Le serveur Django est démarré
  • L'URL est correcte (par défaut: http://localhost:8000)
  • Aucun pare-feu ne bloque la connexion
  • Le port 8000 n'est pas déjà utilisé

Pour démarrer le serveur Django, exécutez :
python manage.py runserver
